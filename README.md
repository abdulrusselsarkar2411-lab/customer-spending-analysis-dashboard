# 📊 Customer Spending & Credit Card Insights Dashboard

📌 End-to-end data analytics project using SQL and Power BI to derive actionable business insights.

---

## 🚀 Project Overview
This project analyzes customer spending behavior to help a bank design effective credit card strategies. It combines SQL for data preparation and Power BI for visualization and insights.

---

## 🔄 Project Workflow
Data Collection → Data Cleaning (SQL) → Data Analysis → Power BI Dashboard → Business Insights

---

## 🛠 Tools Used
- SQL (MySQL)
- Power BI

---

## 📂 Dataset
- dim_customers.csv → Customer demographic data  
- fact_spends.csv → Transaction data  
- customer_analysis.csv → Cleaned & joined dataset used in Power BI  

---

## 🧠 SQL Analysis, Power BI Insights & Business Understanding

### 🔍 SQL Analysis
- Cleaned and validated data by checking for null values and duplicate records
- Joined customer demographic and transaction data to create a unified dataset
- Created key metrics such as Total Spend, Average Spend, and Income Utilization %
- Analyzed spending across categories, cities, payment types, and demographics
- Segmented customers into high, medium, and low value groups

### 📊 Power BI Insights
- Built an interactive dashboard with KPIs: Total Spend, Average Spend, and Income Utilization %
- Visualized spending patterns across categories, cities, and age groups
- Analyzed monthly trends to understand customer behavior over time
- Used slicers for dynamic filtering (City, Gender, Age Group)
- Highlighted key customer segments contributing to higher spending

### 💼 Business Insights & Recommendations
- Customers show high income utilization (~119%), indicating strong credit card potential
- The 25–34 age group contributes the highest spending
- Metro cities like Mumbai and Bengaluru show higher spending activity
- Spending is concentrated in categories like Bills and Groceries
- Credit Card and UPI are the most preferred payment methods

#### 📌 Recommendations:
- Target high-income utilization customers with premium credit cards
- Focus on young professionals in metro cities
- Offer cashback and rewards for high-spend categories
- Use customer segmentation for personalized offers
